## Set up
Cài đặt môi trường poetry với python >=3.9,<3.13,!=3.9.7
```
pip install poetry --user
poetry install
```
Thư viện phù hợp với môi trường được để trong `pyproject.toml`

Để activate môi trường poetry
```poetry shell```
